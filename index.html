CSS Developer Interview Questions

2–3 Years Experience | ~40 Questions

Core Fundamentals

  1. Explain the CSS Box Model. What is the difference between box-sizing: content-box and border-box?
  2. What is the difference between display: none, visibility: hidden, and opacity: 0?
  3. How does CSS specificity work? Calculate the specificity of: #nav .item:hover
  4. What is the difference between em, rem, vw, vh, and px?
  5. What is a stacking context? What properties create a new one?
  6. What is the difference between inline, block, and inline-block elements?
  7. What does the CSS cascade mean? In what order does it resolve conflicts?

Flexbox & Grid

  1. What is the difference between justify-content and align-items in Flexbox?
  2. What does flex: 1 1 auto mean? Explain each value.
  3. What is the difference between flex-grow, flex-shrink, and flex-basis?
  4. How would you create a responsive 3-column grid that collapses to 1 column on mobile without media queries?
  5. What is the difference between auto-fill and auto-fit in CSS Grid?
  6. When would you use CSS Grid over Flexbox?
  7. What is grid-template-areas and how do you use it?
  8. How does align-content differ from align-items in Flexbox?

Positioning

  1. What is the difference between position: relative, absolute, fixed, and sticky?
  2. Why might z-index: 9999 have no effect on an element?
  3. What is the difference between transform: translateX(50%) and left: 50%?
  4. When does position: sticky stop working and why?

Responsive Design

  1. What is the difference between min-width and max-width media queries? Which approach do you prefer?
  2. What is the clamp() function? Give a practical example.
  3. What are CSS container queries and how are they different from media queries?
  4. What is the viewport meta tag and why is it needed?
  5. How would you implement a fluid typography system?

Tricky / Gotchas

  1. What is margin collapsing? When does it happen and how do you prevent it?
  2. Why doesn't vertical-align: middle work on block elements?
  3. What is the difference between :nth-child() and :nth-of-type()?
  4. Why does a parent element collapse when all its children are floated?
  5. Why does text-overflow: ellipsis sometimes not work inside a flex container?
  6. A modal has z-index: 9999 but still appears behind other elements. How would you debug this?

Modern CSS

  1. How do CSS custom properties (variables) differ from SASS variables?
  2. What is the :is() pseudo-class? How does it affect specificity?
  3. What is the difference between :is() and :where()?
  4. What are CSS cascade layers (@layer) and why were they introduced?
  5. What are logical properties in CSS (e.g. margin-inline, padding-block)?
  6. What does the aspect-ratio property do? How was this handled before it existed?

Animations & Performance

  1. Which CSS properties are cheap to animate and which are expensive? Why?
  2. What is the difference between transition and animation in CSS?
  3. What does will-change do and what are the risks of overusing it?
  4. What is the contain property and how can it improve rendering performance?

Scenario Based

  1. SCENARIO: How would you implement a dark mode that respects both system preference and a manual user toggle?
  2. SCENARIO: A sticky header stops working on some pages. What would you investigate?
  3. SCENARIO: You need a responsive image gallery — 4 cols on desktop, 2 on tablet, 1 on mobile — with images maintaining aspect ratio. How do you build it?
  4. SCENARIO: A designer gives you a card with text that must show ellipsis but it's not working inside a flex container. What's the fix?
  5. SCENARIO: You are building a design system. How would you structure your CSS for scalability?
  6. SCENARIO: An animation is causing jank/lag on lower-end devices. How would you diagnose and fix it?
