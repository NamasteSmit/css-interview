Core Fundamentals

  1. Explain the CSS Box Model.
    The CSS box model consists of content, padding, border, and margin. content-box: width does NOT include padding and border. border-box: width includes padding and border (recommended). 
    div {
  width: 200px;
  padding: 20px;
  border: 10px solid;
}
    content-box total width = 260px border-box total width = 200px
  2. display:none vs visibility:hidden vs opacity:0
    display:none removes the element completely from layout. visibility:hidden hides it but keeps space. opacity:0 hides visually but element is still clickable. Example: 
    div { opacity: 0; }
    User can still interact with it.
  3. CSS specificity
    Specificity decides which CSS rule wins. Priority: Inline > ID > Class > Element Example: 
    #nav .item:hover
    Specificity = 1 ID + 2 class = (1,2,0)
  4. Units (em, rem, vw, vh, px)
    px → fixed size em → relative to parent rem → relative to root vw/vh → relative to viewport Example: 
    font-size: 2rem;
  5. Stacking context
    Controls layering of elements. Created by: - position + z-index - transform - opacity < 1 Example: 
    div {
  position: relative;
  z-index: 10;
}
  6. inline vs block vs inline-block
    inline → no width/height block → full width inline-block → supports width/height but stays inline
  7. CSS Cascade
    Order: 1. !important 2. Specificity 3. Source order Example: 
    p { color: red; }
p { color: blue; }
    Blue wins (last rule).

Flexbox & Grid

  1. justify-content vs align-items
    justify-content aligns items horizontally (main axis). align-items aligns vertically (cross axis).
  2. flex: 1 1 auto
    flex-grow:1 → expand flex-shrink:1 → shrink flex-basis:auto → default size
  3. flex properties
    grow expands, shrink reduces, basis defines initial size.
  4. Responsive grid 
    grid-template-columns: repeat(auto-fit, minmax(250px,1fr));
    Auto adjusts columns without media query.
  5. auto-fill vs auto-fit
    auto-fill keeps empty space auto-fit collapses empty tracks
  6. Grid vs Flexbox
    Grid = 2D layouts Flexbox = 1D layouts
  7. grid-template-areas
    Defines layout using names. 
    grid-template-areas:
"header header"
"sidebar main";
  8. align-content vs align-items
    align-items → items align-content → container space

Scenario Based

  1. Dark mode
    Use system + manual toggle. 
    @media (prefers-color-scheme: dark) {
  body { background: black; color: white; }
}
  2. Sticky not working
    Check: - overflow:hidden - parent height - missing top value
  3. Responsive gallery 
    grid-template-columns: repeat(auto-fit, minmax(200px,1fr));
  4. Ellipsis issue 
    min-width: 0;
overflow: hidden;
text-overflow: ellipsis;
  5. Design system
    Use: - BEM - CSS variables - modular structure
  6. Animation lag
    Use transform instead of width/height. 
    transform: translateX(100px);
