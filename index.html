CSS Developer Interview Questions

2–3 Years Experience | Detailed Answers

Core Fundamentals

  1. CSS Box Model
    The box model includes content, padding, border, and margin. content-box → width excludes padding & border border-box → width includes everything (preferred) Example: 
    div {
  width: 200px;
  padding: 20px;
  border: 10px solid;
}
  2. display vs visibility vs opacity
    display:none removes element from layout. visibility:hidden keeps space but hides it. opacity:0 hides visually but element is still interactive.
  3. Specificity
    Order: Inline > ID > Class > Element #nav .item:hover = (1,2,0)
  4. Units
    px fixed, em parent-based, rem root-based, vw/vh viewport-based.
  5. Stacking Context
    Controls layering. Created by position + z-index, transform, opacity, etc.
  6. inline vs block vs inline-block
    inline → no width/height block → full width inline-block → supports size but inline behavior
  7. CSS Cascade
    Priority: !important > specificity > source order.

Flexbox & Grid

  1. justify-content vs align-items
    justify-content controls main axis, align-items controls cross axis.
  2. flex:1 1 auto
    grow = 1, shrink = 1, basis = auto.
  3. flex properties
    grow expands, shrink reduces, basis defines initial size.
  4. Responsive grid 
    grid-template-columns: repeat(auto-fit, minmax(250px,1fr));
  5. auto-fill vs auto-fit
    auto-fill keeps empty tracks, auto-fit collapses them.
  6. Grid vs Flexbox
    Grid is 2D, Flexbox is 1D.
  7. grid-template-areas
    Defines layout using names. 
    grid-template-areas:
"header header"
"sidebar main";
  8. align-content vs align-items
    align-items → individual items align-content → entire grid/flex lines

Positioning

  1. position types
    relative, absolute, fixed, sticky. sticky example: 
    position: sticky;
top: 0;
  2. z-index issue
    Requires positioned element. Also affected by stacking context.
  3. translate vs left
    transform is GPU accelerated (better performance), left causes reflow.
  4. sticky not working
    Fails if parent has overflow hidden or no scroll space.

Responsive Design

  1. min-width vs max-width
    min-width → mobile-first (recommended). max-width → desktop-first.
  2. clamp() 
    font-size: clamp(16px, 2vw, 24px);
    Responsive sizing without media queries.
  3. Container queries
    Respond to parent size instead of viewport.
  4. viewport meta
    Ensures proper scaling on mobile devices.
  5. Fluid typography
    Use clamp() or vw-based sizing.

Tricky

  1. Margin collapsing
    Vertical margins merge. Fix with padding or overflow.
  2. vertical-align
    Works only on inline/table elements.
  3. nth-child vs nth-of-type
    nth-child counts all elements, nth-of-type counts specific type.
  4. Float collapse
    Parent collapses because floats are removed from normal flow. Fix with clearfix or overflow.
  5. ellipsis not working
    Needs: 
    overflow: hidden;
white-space: nowrap;
text-overflow: ellipsis;
min-width: 0;
  6. Modal behind elements
    Check stacking context and parent transforms.

Modern CSS

  1. CSS vs SASS variables
    CSS variables are dynamic, SASS variables are compile-time.
  2. :is()
    Simplifies selectors while keeping specificity.
  3. :is() vs :where()
    :is keeps specificity, :where removes it.
  4. @layer
    Controls cascade order.
  5. Logical properties
    margin-inline replaces left/right (RTL friendly).
  6. aspect-ratio
    Maintains element ratio. Previously done using padding hack.

Performance

  1. Cheap vs expensive animations
    Cheap → transform, opacity Expensive → width, height, top
  2. transition vs animation
    transition = state change animation = keyframes
  3. will-change
    Optimizes rendering but overuse increases memory usage.
  4. contain
    Limits layout/paint scope → improves performance.

Scenario Based

  1. Dark mode
    Use prefers-color-scheme + manual toggle.
  2. Sticky debug
    Check overflow, parent height, scroll container.
  3. Responsive gallery
    Use grid with auto-fit and minmax.
  4. Ellipsis issue
    Add min-width:0 in flex container.
  5. Design system
    Use modular CSS, variables, naming conventions like BEM.
  6. Animation lag
    Use transform, reduce reflow, optimize repaint.
